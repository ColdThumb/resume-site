私について

新しいことに挑戦し続け、創造力と学習意欲にあふれたエンジニア

1999年に南の島で生まれ、18年間をそこで過ごした後、進学と就職のために2200km離れた都市へ移り住みました。 さらに多様な経験を求め、2022年10月に来日し、語学学校で学びながら生活を始めました。 幼い頃から日本のアニメ文化に親しみ、その影響で自然と日本に惹かれるようになりました。しかし、日本に長く住みたいと考える最も大きな理由は、暮らしやすさにあります。人々が互いに程よい距離感を保ち、ルールを尊重する社会の中で生活することで、心の平穏を得ることができています。 日常では新しい技術の習得、日英の語学学習に励み、常に成長を意識しています。右のプロフィールには「フルスタック開発者」と記していますが、まだまだ学ぶべきことは多く、今後も努力を重ねていきたいと思っています。

実績

経験と技術力を活かし、クライアントのニーズに応えるソリューションを提供してきました。

業務システム開発と改善

新規画面の設計・開発や不具合修正を通じて業務効率化・安定稼働に貢献しました。DBとアプリ間のマッピング不整合も解消し、データの整合性を確保しています。

フルスタックなWebアプリ開発経験

フロントエンドとバックエンドを分離したアーキテクチャで、Rails・PostgreSQL を基盤とした RESTful API の設計・実装を担当。認証、権限管理、メール送信など実務で必要とされる機能を自ら構築しました。

要件定義から運用までの一貫した開発サイクル

個人開発では、要件定義・設計・実装・運用をすべて一人で担当。実務での経験に加え、企画から運用まで全体を見通す力を養い、開発サイクルを通して課題解決力を培いました。
